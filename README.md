# Backend Development Repository

This repository contains my backend development learning projects built with Node.js, Express, and MongoDB. It serves as a space where I practice and improve my backend skills by building RESTful APIs, working with databases, and implementing authentication systems.

## 🚀 Purpose
- Learn backend development fundamentals
- Build RESTful APIs using Node.js and Express
- Work with MongoDB and Mongoose
- Understand authentication, middleware, and server-side logic
- Practice real-world backend project structure

## 🛠️ Tech Stack
- Node.js  
- Express.js  
- MongoDB  
- Mongoose  
- JSON Web Tokens (JWT)

## 📂 Projects Included
- Task Manager API (CRUD operations, authentication, protected routes)
- Notes App API (CRUD operations, authentication, user-specific notes, protected routes)

## 🎯 Goal
To strengthen backend development skills by building practical and scalable API projects.
